## Project Name

Automated File Organizer

## Project Description

This project automatically organizes files into different folders based on their file extensions. It helps users keep their directories clean and well organized.

## Features

* Graphical User Interface (GUI)
* Select any folder from the computer
* Automatically creates category folders
* Organizes files into:

  * Images
  * Documents
  * Videos
  * Audio
  * Archives
  * Others
* Easy to use

## Technologies Used

* Python 3.11
* Tkinter
* OS Module

## How to Run

1. Open the project folder.
2. Run:

python organizer.py

3. Select a folder.
4. Files will be organized automatically.

## Project Outcome

The project successfully sorts files into their respective categories and improves file management efficiency.
